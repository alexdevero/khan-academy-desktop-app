### Install

#### Clone this repo

```
git clone https://github.com/alexdevero/khan-academy-desktop-app.git
```

#### Install dependencies

```
npm install
```
or
```
yarn
```

### Usage

#### Run the app

```
npm run start
```
or
```
yarn run start
```

#### Build the app

```
npm run package
```
or
```
yarn run package
```
